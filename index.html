ScriptFodder

A marketplace for Garry's Mod content creators and server owners to buy and sell addons relating to Garry's Mod and other Source games. The site handles tens of thousands of dollars worth of transactions every single month and is the single biggest website of its kind.

I developed and created the entire stack from scratch to handle the large amount of users whilst still being able to defend against DDoS attacks - the stack consists of multiple servers with full automatic load balancing using a combination of CloudFlare, nginx, PHP 7, MySQL, Redis and a whole bunch more.

Don't Starve Modding Tools

A project I started in late 2012 following the release of Don't Starve. The set of tools provided a way for modders to import/export game textures to the game allowing for a variety of different mods to be created. Later with support from the games developers we devised a set of official tools distributed through Steam. The game is now one of the few to use Steam Workshop to allow distribution of mods.
